# 🚀 eShop - (React - Html - Css) 🚀

Proyecto creado en REACT para manejar un eShop.

## 🛠️ Instalación

Para utilizar este proyecto primero instale las dependencias:

```bash
npm install
```

## 🕹 Uso

```bash
npm run start
```
